# Project Arcade Manager
Arcaea自制工具【Arcade】的发行版管理器

### 程序特点
- 可以自动搜索已安装的Arcade发行版
- 可以实时同步已安装的Arcade发行版中的通用数据(如谱面游玩背景)

### 已支持的Arcade发行版
- Arcade
- Arcade-Plus
- Arcade-Zero
- *Arcade-Alpha*(不完全支持)
- *Arcade-One*(不完全支持)
