# Project Arcade Hub
Arcaea自制工具【Arcade】的发行版管理器

### 程序特点
- 系统化管理Arcade项目(仅支持Arcade-Chan发行版)
- 实时同步已安装的Arcade发行版中的通用数据(如谱面游玩背景)

### 已支持的Arcade发行版
- Arcade
- Arcade-Chan
- Arcade-One
- Arcade-Plus
- Arcade-Zero
- *Arcade-Alpha*(不完全支持)
